# 📚 Personal Knowledge Base RAG System

A **Retrieval-Augmented Generation (RAG)** application built with Streamlit, Chroma, HuggingFace embeddings, and DeepSeek API.  
Upload your documents (PDF, TXT, Markdown) and ask questions – the AI answers based **only** on your documents.

## ✨ Features

- 📄 **Document ingestion** – Upload PDF, TXT, or Markdown files, or pick existing files from `./library` folder.
- 🔍 **Semantic search** – Uses `sentence-transformers/all-MiniLM-L6-v2` to embed documents and queries.
- 🧠 **RAG answers** – Answers are generated by DeepSeek API using only the retrieved relevant chunks.
- 💾 **Persistent vector store** – Documents are stored locally in `./chroma_db`; no need to re‑embed after restart.
- 🔄 **Reprocess local files** – If a file in `./library` changes, you can delete its old vectors and reprocess it with one click.
- ✍️ **Query rewriting** – Vague queries like “file content” are rewritten into more effective retrieval queries.
- 📋 **Processed file list** – Shows all files already in the knowledge base, with the option to reprocess local ones.
- 🚫 **Automatic deduplication** – Prevents adding the same file twice (based on file ID).

## 🧰 Requirements

- Python 3.9 or higher
- A [DeepSeek API key](https://platform.deepseek.com/) (free tier available)





# 📚 个人知识库 RAG 系统

基于 Streamlit、Chroma、HuggingFace 嵌入模型和 DeepSeek API 构建的**检索增强生成（RAG）**应用。  
上传你的文档（PDF、TXT、Markdown），然后提问——AI 将**仅根据**你的文档内容进行回答。

## ✨ 功能特性

- 📄 **文档导入** – 上传 PDF、TXT 或 Markdown 文件，或从 `./library` 文件夹中选择已有文件。
- 🔍 **语义检索** – 使用 `sentence-transformers/all-MiniLM-L6-v2` 对文档和查询进行向量化。
- 🧠 **RAG 问答** – 由 DeepSeek API 基于检索到的相关片段生成答案。
- 💾 **持久化向量库** – 文档存储在本地 `./chroma_db` 中，重启后无需重新嵌入。
- 🔄 **重新处理本地文件** – 如果 `./library` 中的文件发生变化，可以一键删除旧向量并重新处理。
- ✍️ **查询改写** – 将模糊查询（如“文件内容”）改写成更有效的检索查询。
- 📋 **已处理文件列表** – 显示知识库中的所有文件，并支持重新处理本地文件。
- 🚫 **自动去重** – 基于文件 ID 防止重复添加同一文件。

## 🧰 环境要求

- Python 3.9 或更高版本
- [DeepSeek API 密钥](https://platform.deepseek.com/)（免费额度可用）
